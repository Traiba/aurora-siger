# aurora-siger
O projeto Aurora Siger simula a decolagem de uma nave para Marte, integrando computação embarcada, Python, IA e eficiência energética. O desafio é criar um Relatório Operacional que valide a telemetria (pressão, calor, energia) via algoritmos, garantindo decisões autônomas e éticas em um cenário de alta complexidade aeroespacial.
